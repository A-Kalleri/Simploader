# Simploader

A small Chrome extension for saving and restoring browser sessions.

Simploader saves all tabs from the current browser window into a single file, making it easy to restore a session later.

## Features

- Save all tabs from the current window
- Restore saved sessions with a few clicks
- Sessions are stored as `.sld` files with basic XOR obfuscation
- Supports incognito windows
- Lightweight and simple
